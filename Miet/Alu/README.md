 Файл alu.sv содержит логику работы АЛУ, в не осуществляется работа команд, пакет параметров с опкодами которых представлен в файле alu_opcodes_pkg.sv
 Также в АЛУ используется ранее описанный модуль 32 битного сумматора из 1 лабораторноый работы

 Для запуска тестбенча необходимо иметь платную версию Quartus, возможно бесплатную версию Vivado, в версии Quartus Prime Lite Edition тетсбенч не проходит задачу
на 133 строчке из за использования .randomize()
