 Файл alu.sv содержит логику работы АЛУ, в нем осуществляется работа команд, пакет параметров с опкодами которых представлен в файле alu_opcodes_pkg.sv
 
 Также в АЛУ используется ранее описанный модуль 32 битного сумматора из 1 лабораторной работы

 Для запуска тестбенча необходимо иметь платную версию Quartus, возможно бесплатную версию Vivado, в версии Quartus Prime Lite Edition 21.1.1 тетсбенч не проходит задачу на 133 строчке из за использования .randomize()
